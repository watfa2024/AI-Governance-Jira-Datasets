# AI-Governance-Jira-Datasets
Public Jira Repository Datasets for AI Governance Readiness Research  This repository contains two publicly available Jira software engineering datasets used in the study:  From Software Repositories to Ethical AI: AI Governance Readiness Emergence Theory (AGRET) for Software Project Management
1- https://zenodo.org/records/7022735
2- https://zenodo.org/records/4459154

Purpose

These datasets were used to develop and validate the AI Governance Readiness Emergence Theory (AGRET) and the proposed AI Governance Readiness Index (AGRI).

Within our study, historical Jira repository information was transformed into Repository Governance Indicators (RGIs), including:

Governance Transparency
Governance Accountability
Workflow Integrity
Governance Fairness
Estimation Reliability
Human Oversight

These indicators were subsequently analyzed using:

Repository Mining
Machine Learning
Explainable Artificial Intelligence (SHAP)
Structural Equation Modeling (SEM)
Robustness and Sensitivity Analysis
